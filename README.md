# Konsept Fullstack-webapplikasjon

Navn: "Infoplay" Salgsside for plug-and-play infoskjerm.

Målgruppe: Småbedrifter, kafeer, frisører.

Hovedfunksjon: Vis produkt, pris, demo, og samle leads via kontaktform.

Teknologi (modul 1+2): Postgres i Docker, SQL-skript, statisk frontend med HTML/CSS/JS.

Levering dag 1: Plan.md med kort beskrivelse av konsept og teknologivalg.
